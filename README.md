# Geometric-HyperLoRA
Hypernetwork-generated subspaces with geometric control, aiming for zero-shot task adaptation.

We have discovered that task adaptation is fundamentally geometric. By learning to generate adaptation subspaces directly from task descriptions, we achieve zero-shot generalization that outperforms both random baselines and naive transfer. This opens a new paradigm: geometric model adaptation where tasks are represented as subspaces in the model's latent space.
